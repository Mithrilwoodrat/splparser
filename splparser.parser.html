splparser.parser¶

SPLParser.__init__(lexermod, parsetab_name, parsetab_dir, logname, rulesmod, optimize=True)¶

Creates an SPLParser object.

Parameters:
  • self (SPLParser) – The object being created
  • lexermod (module) – The corresponding lexer module
  • parsetab_name (str) – The name of the toplevel parse table file
  • parsetab_dir (str) – The directory in which to store parse tables
  • logname (str) – The name of the log
  • rulesmod (module) – The module containing the top-level rules
  • optimize (dir) – Whether or not to write a log (it slows performance and can cause the parser to sometimes fail due to too many open file handles – one for each command parsed as they have separate log files)
Return type:

SPLParser
class splparser.parser.SPLParser(lexermod, parsetab_name, parsetab_dir, logname, rulesmod, optimize=True)¶

Represents a parser object that can parse SPL queries.
